HTML5

HTML (Lenguaje de Marcado de Hipertexto) es un lenguaje de programación que se utiliza para estructurar y desplegar el contenido de una página web. HTML es el componente más básico de la web y define el significado y la estructura del contenido web.

CSS3

CSS son las siglas de "Cascading Style Sheets" (hojas de estilo en cascada). Es un lenguaje de programación que determina el diseño de los documentos electrónicos

Diseño Grafico

El diseño gráfico es una disciplina que combina texto e imágenes para transmitir un mensaje visual. Se utiliza principalmente en marketing y comunicación corporativa, como en el diseño de páginas web
